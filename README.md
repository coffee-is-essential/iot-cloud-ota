# Overview

대규모 IoT 단말 관리를 위한 클라우드 기반 OTA 기술

# How to run

## Frontend
Frontend는 Typescript기반 React를 사용합니다.
또한, Package Manager로 `pnpm`, 빌드 툴로 `vite`를 사용합니다.
로컬에서 개발 모드를 실행하기 위해서는 다음 명령어를 실행하면 됩니다.

```
cd frontend
pnpm install
pnpm dev
```

만약 `pnpm`이 로컬에 설치되어 있지 않다면 다음 명령어를 실행하여 `pnpm`을 설치합니다.
```
npm install -g pnpm
```
